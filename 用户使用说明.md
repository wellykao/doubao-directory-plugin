# 豆包对话目录导航插件 - 用户使用说明

## 📖 插件简介

豆包对话目录导航插件是一款专为豆包AI对话设计的浏览器扩展，能够自动识别并整理你的对话内容，生成清晰的目录列表。通过这个插件，你可以快速定位到历史对话中的任意位置，大大提升长对话的浏览效率。

## ✨ 主要功能

### 1. 自动生成对话目录
- 自动识别豆包对话中的用户提问
- 实时更新目录内容
- 智能过滤文件上传记录，只显示真实问题

### 2. 快速定位
- 点击目录项，自动滚动到对应的对话位置
- 高亮显示目标对话，方便查看

### 3. 悬停查看完整内容
- 鼠标悬停在目录项上，显示完整的提问内容
- 智能判断显示位置，避免遮挡

### 4. 灵活的界面控制
- **最小化**：只显示标题栏，节省空间
- **最大化**：恢复完整显示
- **关闭**：完全关闭插件，停止运行
- **拖动**：可以拖动到屏幕任意位置
- **调整大小**：拖动右下角调整面板尺寸

### 5. 快捷键支持
- `Ctrl + Shift + D`：快速显示/隐藏插件

## 📦 安装步骤（超详细版）

### 第一步：下载插件文件

1. 找到插件文件夹，里面应该包含以下文件：
   - `manifest.json`（配置文件）
   - `content.js`（功能脚本）
   - `styles.css`（样式文件）
   - `icons/`（图标文件夹）
   - `用户使用说明.md`（本文件）

2. 确保所有文件都在同一个文件夹内，不要分开存放

### 第二步：打开浏览器扩展管理页面

#### Chrome/Edge 浏览器：
1. 在浏览器地址栏输入：`chrome://extensions/`
2. 按回车键进入扩展管理页面

#### Firefox 浏览器：
1. 在浏览器地址栏输入：`about:addons`
2. 按回车键进入扩展管理页面

### 第三步：启用开发者模式（重要！）

#### Chrome/Edge 浏览器：
1. 在扩展管理页面的右上角，找到"开发者模式"开关
2. 点击开关，使其变为开启状态（蓝色）
3. 开启后会显示更多的选项和按钮

#### Firefox 浏览器：
1. 点击页面右上角的齿轮图标（设置）
2. 在弹出的菜单中勾选"调试附加组件"

### 第四步：加载插件

#### Chrome/Edge 浏览器：
1. 在扩展管理页面，点击左上角的"加载已解压的扩展程序"按钮
2. 在弹出的文件选择窗口中，找到并选中插件文件夹
3. 点击"选择文件夹"按钮
4. 如果看到安全提示，点击"继续"或"允许"
5. 插件安装成功后，会在扩展列表中显示

#### Firefox 浏览器：
1. 点击页面上的"临时附加组件"按钮
2. 在弹出的文件选择窗口中，找到并选中插件文件夹
3. 点击"打开"按钮
4. 点击"添加"确认安装

### 第五步：验证安装

1. 打开豆包网站：https://www.doubao.com/
2. 开始一段对话
3. 观察页面右侧是否出现了"对话目录"面板
4. 如果出现，说明安装成功！

## 🎯 使用方法

### 基本使用

1. **自动生成目录**
   - 打开豆包对话页面
   - 插件会自动识别对话内容并生成目录
   - 目录会实时更新，无需手动操作

2. **快速定位**
   - 点击目录中的任意一项
   - 页面会自动滚动到对应的对话位置
   - 目标对话会高亮显示3秒

3. **查看完整内容**
   - 将鼠标悬停在目录项上
   - 会显示完整的提问内容
   - tooltip会智能选择显示位置

### 界面操作

1. **最小化面板**
   - 点击标题栏上的"−"按钮
   - 面板会收缩为只显示标题栏
   - 再次点击"□"按钮可以恢复

2. **关闭插件**
   - 点击标题栏上的"×"按钮
   - 插件会完全停止运行
   - 按`Ctrl + Shift + D`可以重新启动

3. **拖动位置**
   - 按住标题栏（不要按按钮）
   - 拖动到想要的位置
   - 松开鼠标即可

4. **调整大小**
   - 将鼠标移动到面板右下角
   - 当鼠标变成斜向箭头时，按住拖动
   - 调整到合适的大小后松开

### 快捷键

- `Ctrl + Shift + D`：显示/隐藏插件
  - 如果插件已关闭，会重新启动
  - 如果插件正在运行，会完全关闭

## ❓ 常见问题

### Q1: 安装后没有看到目录面板怎么办？

**A:** 请尝试以下方法：
1. 刷新豆包页面
2. 确认插件在扩展管理页面中已启用
3. 检查是否在豆包网站上（插件只在豆包网站生效）
4. 尝试关闭并重新加载插件

### Q2: 目录显示的内容不完整？

**A:** 这可能是以下原因：
1. 对话内容还在加载中，请稍等片刻
2. 插件过滤了文件上传记录，这是正常行为
3. 尝试滚动页面，让所有内容加载完成

### Q3: 点击目录项没有反应？

**A:** 请检查：
1. 确认点击的是目录项，而不是其他位置
2. 查看浏览器控制台是否有错误信息
3. 尝试刷新页面重新加载插件

### Q4: 面板位置或大小不对？

**A:** 你可以：
1. 拖动标题栏调整位置
2. 拖动右下角调整大小
3. 如果位置完全错误，关闭插件后重新打开

### Q5: 插件影响页面性能吗？

**A:** 插件经过优化，对性能影响很小：
1. 使用高效的DOM观察机制
2. 智能去重，避免重复处理
3. 可以随时关闭插件停止运行

### Q6: 如何卸载插件？

**A:** 卸载步骤：
1. 打开扩展管理页面
2. 找到"豆包对话目录导航"插件
3. 点击"移除"或"删除"按钮
4. 确认删除

## ⚠️ 注意事项

1. **只在豆包网站生效**
   - 插件只在豆包网站上工作
   - 其他网站不会受到影响

2. **隐私安全**
   - 插件只在本地运行
   - 不会上传任何对话内容
   - 不会收集用户数据

3. **兼容性**
   - 支持Chrome、Edge、Firefox等主流浏览器
   - 需要浏览器支持Manifest V3

4. **更新建议**
   - 定期检查插件更新
   - 更新前建议备份重要对话

5. **使用建议**
   - 长对话时使用效果最佳
   - 可以随时关闭插件节省资源
   - 建议配合豆包的搜索功能使用
