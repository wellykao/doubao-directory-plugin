## 📖 项目简介

豆包对话目录导航插件是一款智能的浏览器扩展，能够自动识别并整理豆包AI对话中的用户提问，生成清晰的目录列表。通过点击目录项，用户可以快速定位到历史对话中的任意位置，大大提升长对话的浏览效率。

### 核心价值

- 🚀 **提升效率**：快速定位历史对话，无需手动滚动
- 🎯 **精准导航**：智能识别用户提问，过滤无关内容
- 💡 **智能交互**：悬停查看完整内容，智能判断显示位置
- 🎨 **灵活界面**：支持拖动、缩放、最小化等操作
- 🔒 **隐私安全**：本地运行，不上传任何数据

## 📦 安装指南

### 前置要求

- Chrome 88+ / Edge 88+ / Firefox 78+
- 支持Manifest V3的浏览器

### 安装步骤

#### 方法一：从源码安装（推荐）

1. **下载插件**
   ```bash
   git clone https://github.com/wellykao/doubao-directory-plugin.git
   cd doubao-directory-plugin
   ```

2. **打开浏览器扩展管理页面**
   - Chrome/Edge: 访问 `chrome://extensions/`
   - Firefox: 访问 `about:addons`

3. **启用开发者模式**
   - Chrome/Edge: 点击右上角的"开发者模式"开关
   - Firefox: 点击齿轮图标，勾选"调试附加组件"

4. **加载插件**
   - Chrome/Edge: 点击"加载已解压的扩展程序"，选择插件文件夹
   - Firefox: 点击"临时附加组件"，选择插件文件夹

5. **验证安装**
   - 访问豆包网站：https://www.doubao.com/
   - 观察页面是否出现"对话目录"面板

#### 方法二：从Chrome Web Store安装（即将上线）

## 🎯 使用说明

### 基本使用

1. **自动生成目录**
   - 打开豆包对话页面
   - 插件自动识别对话内容并生成目录
   - 目录会实时更新，无需手动操作

2. **快速定位**
   - 点击目录中的任意一项
   - 页面自动滚动到对应的对话位置
   - 目标对话高亮显示3秒

3. **查看完整内容**
   - 将鼠标悬停在目录项上
   - 显示完整的提问内容
   - tooltip智能选择显示位置

### 界面操作

| 操作 | 说明 |
|------|------|
| **最小化** | 点击"−"按钮，只显示标题栏 |
| **最大化** | 点击"□"按钮，恢复完整显示 |
| **关闭** | 点击"×"按钮，完全关闭插件 |
| **拖动** | 按住标题栏拖动到任意位置 |
| **调整大小** | 拖动右下角调整面板尺寸 |

### 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl + Shift + D` | 显示/隐藏插件 |

- **前端技术**: JavaScript (ES6+), CSS3

## ❓ 常见问题

### Q: 插件只在豆包网站生效吗？
**A:** 是的，插件专为豆包网站设计，只在豆包网站上工作。

### Q: 插件会收集我的对话数据吗？
**A:** 不会。插件完全在本地运行，不会上传任何对话内容或用户数据。

### Q: 如何更新插件？
**A:** 如果是从源码安装，需要重新下载最新代码并重新加载插件。

### Q: 插件支持哪些浏览器？
**A:** 目前支持Chrome、Edge、Firefox等主流浏览器，需要支持Manifest V3。

### Q: 如何卸载插件？
**A:** 在浏览器扩展管理页面找到插件，点击"移除"或"删除"按钮即可。

更多问题请查看[用户使用说明.md](用户使用说明.md)。
